## Frontend MCP-Hub – Version 1

### Exécution locale :

- Installer les dépendances:
  ```bash
  npm install
  ```
- Démarrer le serveur de dev:
  ```bash
  npm run dev
  ```
- Build de production:
  ```bash
  npm run build
  ```
- Prévisualisation du build:
  ```bash
  npm run preview
  ```

### Docker

- Construire et lancer via docker-compose:
  ```bash
  docker compose up --build
  ```
